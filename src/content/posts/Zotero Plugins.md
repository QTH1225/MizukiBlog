# Zotero插件详细指南

## 概述

Zotero是一款强大的开源文献管理工具，通过安装各种插件可以极大地扩展其功能。本文详细介绍博主使用的Zotero插件，包括功能说明、安装方法、使用技巧和实际应用案例。

## Zotero 7.0.13

这是Zotero的主程序版本。7.0版本是Zotero的重大更新，从XUL架构转向了WebExtension，带来了性能提升和更好的兼容性。64位版本可以更好地利用内存，适合处理大量的参考文献和PDF文件。

**下载链接：**
- [Zotero官网下载](https://www.zotero.org/)
- [Zotero 中文社区](https://zotero-chinese.com/)
- [PKMer_Zotero 介绍](https://pkmer.cn/Pkmer-Docs/11-zotero/zotero/)

**安装方法：**
1. 下载对应操作系统的安装包
2. 运行安装程序，按照提示完成安装
3. 首次启动时配置同步账户（可选）
4. 安装浏览器连接器以便从网页抓取文献

## 插件详细说明
下面详细介绍博主使用的Zotero插件。这些插件在文献管理和研究中发挥了重要作用，帮助用户更高效地组织和利用文献资源。
### 插件安装通用方法
所有插件安装方法类似：
1. 下载插件xpi文件
2. 打开Zotero → 工具 → 插件 → 安装插件
3. 选择xpi文件并安装
4. 重启Zotero生效

### 1. Action and Tags for Zotero 2.1.1
Action and Tags插件是一个强大的标签管理和批量操作工具，它允许用户基于标签执行各种自动化操作。该插件提供了批量标签管理功能，可以快速为多个文献条目添加、删除或修改标签，同时支持条件操作，根据标签条件自动执行如移动文献到特定文件夹等操作。智能标签功能能够基于文献内容自动生成相关标签，而标签组管理则便于用户对标签进行分类管理，大大提高了文献组织的效率。

**使用：** 右键文献选择"标签操作"进行批量处理
**兼容性：** Zotero 6.x（Zotero 7需检查更新）
**应用：** 自动分类文献、批量标签管理

### 2. Add-on Market for Zotero 2.1.1
Add-on Market是一个内置的插件市场，让用户可以直接在Zotero界面中浏览、安装、更新和管理插件。这个插件极大地简化了插件的获取过程，用户无需手动下载xpi文件，只需在市场中点击安装即可完成插件部署。它能够自动检测插件更新并提供一键更新功能，同时还支持查看已安装插件的状态和启用/禁用管理。

**使用：** 工具 → 插件 → 市场标签页
**兼容性：** Zotero 6.x/7.x
**应用：** 快速发现和安装新插件

### 3. Better Notes for Zotero 2.5.1
Better Notes是一个强大的笔记管理插件，提供比Zotero原生笔记更丰富的功能。它完全支持Markdown语法，用户可以使用标准的Markdown标记来编写格式化的笔记内容。插件内置了模板系统，用户可以创建和使用笔记模板，大大提高笔记编写效率。笔记关联功能允许将笔记与特定文献条目关联，而笔记组织功能则支持创建笔记文件夹和层级结构，便于管理大量的研究笔记。

**使用：** 右键文献创建Better Notes
**兼容性：** Zotero 7.x
**应用：** 文献阅读笔记、研究想法记录

### 4. Del Item with Attachment 0.4.0
Del Item with Attachment插件在删除文献条目时自动删除相关的附件文件，避免产生孤立的文件占用存储空间。这个插件解决了Zotero原生删除功能的一个痛点，即删除文献条目后附件文件仍然保留在存储目录中。它提供了安全确认对话框，防止用户误删重要文件，同时支持批量处理功能，可以一次性删除多个文献条目及其关联附件。

**使用：** 右键选择"删除条目和附件"
**兼容性：** Zotero 6.x（Zotero 7可能已集成）
**应用：** 清理重复文献、管理存储空间

### 5. Easier Citation 0.5.0
Easier Citation插件简化了文献引用过程，提供更快捷的引用插入方式。它通过快捷键或右键菜单快速插入引用，在插入前可以预览引用格式，确保引用样式符合要求。插件支持批量引用功能，用户可以一次性插入多个文献引用，大大提高了学术写作的效率。自定义快捷键功能允许用户设置个性化的引用快捷键，进一步优化工作流程。

**使用：** 快捷键或右键菜单插入引用
**兼容性：** Zotero 6.x（需配合文字处理软件）
**应用：** 论文写作、批量引用管理

### 6. Ethereal Reference 1.4.5
Ethereal Reference专注于美化参考文献的显示和导出格式，提供更专业的引用样式。该插件优化了参考文献的视觉呈现，用户可以创建个性化的引用样式，满足不同学术期刊的格式要求。它提供了多种学术期刊的引用格式适配，同时改善了参考文献导出的格式和质量，确保最终输出的参考文献具有专业的外观和标准的格式。

**使用：** 引用样式管理器选择Ethereal样式
**兼容性：** Zotero 6.x/7.x
**应用：** 学术论文格式化、期刊适配

### 7. Ethereal Style 5.6.2
Ethereal Style是Zotero的界面美化插件，提供丰富的主题和自定义选项。它支持多种颜色主题和界面风格的切换，用户可以根据个人喜好选择不同的视觉样式。插件允许自定义界面字体和大小，调整Zotero界面元素的布局，同时还提供暗色模式支持，保护用户在长时间使用时的视力健康。这些美化功能不仅提升了用户体验，也使得研究工作环境更加个性化。

**使用：** 设置中配置Ethereal Style选项
**兼容性：** Zotero 7.x
**应用：** 个性化界面、护眼主题

### 8. Jasminum 1.1.17
Jasminum是专为中文用户设计的插件，特别优化了中文文献的识别和管理。它能够自动识别和抓取知网文献的元数据，优化中文文献的标题、作者名显示格式。插件根据中文规则智能重命名PDF附件，支持批量处理中文文献，大大提高了中文文献管理的效率。对于主要使用中文数据库的研究人员来说，这个插件是必不可少的工具。

**使用：** 导入中文文献后右键优化
**兼容性：** Zotero 6.x（Zotero 7部分功能受限）
**应用：** 中文文献管理、知网论文导入

### 9. Linter for Zotero 1.24.5
Linter插件用于检查和修复文献条目的元数据问题，确保文献库的准确性和一致性。它能够扫描文献条目的元数据完整性，识别可能的重复条目，并提供一键修复常见的元数据问题功能。插件还支持格式标准化，统一作者名、标题等格式，帮助用户维护一个整洁规范的文献数据库。定期运行Linter可以保持文献库的良好状态。

**使用：** 右键选择Linter检查元数据
**兼容性：** Zotero 6.x/7.x
**应用：** 定期文献库维护、元数据修复

### 10. Sci-PDF 1.3.0
Sci-PDF专注于科学PDF文件的处理，提供增强的PDF阅读和内容提取功能。该插件改善了PDF阅读体验，提供更好的标注工具和内容提取能力。它能够自动提取PDF中的图表、公式等重要内容，识别PDF中的参考文献列表，并将这些信息整合到Zotero中。增强的PDF标注功能使得用户可以更有效地管理阅读笔记和重要信息。

**使用：** 打开PDF使用增强工具
**兼容性：** Zotero 6.x
**应用：** 科学PDF处理、内容提取

### 11. Translate for Zotero 2.2.18
Translate插件提供文献内容的翻译功能，帮助用户阅读外文文献。它支持翻译文献标题和摘要内容为中文，提供多语言翻译支持，用户可以一次性翻译多个文献。这个插件特别适合需要阅读大量外文文献的研究人员，能够快速了解文献的主要内容，提高阅读效率。需要注意的是，使用翻译功能需要稳定的网络连接。

**使用：** 右键选择翻译功能
**兼容性：** Zotero 6.x（需网络连接）
**应用：** 外文文献阅读、批量翻译

### 12. Zoplicate 3.0.8
Zoplicate专门用于检测和删除重复的文献条目，保持文献库的整洁。它采用智能算法识别重复的文献条目，提供多种合并重复条目的方式，支持批量检测和删除重复。用户可以设置重复检测的敏感度，根据个人需求调整检测规则。定期使用Zoplicate可以有效地清理从多个数据库导入的重复文献，保持文献库的高效运行。

**使用：** 运行扫描检测重复条目
**兼容性：** Zotero 6.x/7.x
**应用：** 文献库清理、重复合并

### 13. Zotero Attanger 1.3.5
Zotero Attanger是附件管理工具，提供强大的附件组织和处理功能。它支持批量重命名附件文件，整理附件的存储位置，进行附件格式转换，并提供附件使用情况统计。这个插件解决了Zotero在附件管理方面的不足，使得用户可以更有效地管理大量的PDF和其他附件文件，保持附件命名的标准化和存储结构的合理性。

**使用：** 选择文献使用Attanger功能
**兼容性：** Zotero 6.x（Zotero 7部分功能集成）
**应用：** 附件标准化、批量处理

### 14. Zotero Better Author 5.0.0
Better Author插件优化作者名的显示和格式化，提供更专业的作者信息管理。它能够统一作者名的显示格式，智能解析复杂的作者名格式，支持批量修改作者名格式。用户可以创建个性化的作者名格式化规则，特别适合处理包含特殊字符的多语言作者名。这个插件有助于提高参考文献的专业性和一致性。

**使用：** 配置偏好后应用格式化
**兼容性：** Zotero 6.x/7.x
**应用：** 作者名标准化、多语言处理

### 15. Zotero Debug Bridge 1.0
Debug Bridge是开发工具，主要用于调试Zotero和插件的问题。它提供详细的调试日志和信息，帮助定位插件冲突和错误，分析Zotero的性能问题。这个插件主要面向开发者和高级用户，为插件开发提供调试支持。当遇到插件冲突或兼容性问题时，启用Debug Bridge可以帮助快速定位问题原因。

**使用：** 主要用于开发调试场景
**兼容性：** Zotero 6.x（面向开发者）
**应用：** 插件调试、性能优化

### 16. Zotero Reading List 1.5.10
Reading List插件帮助用户创建和管理阅读清单，规划阅读进度。它可以创建待阅读文献的清单，标记文献的阅读状态，为阅读任务设置优先级，并提供提醒功能和时间安排。这个插件特别适合需要管理大量阅读材料的研究人员和学生，能够有效地跟踪学术论文的阅读进度，规划研究项目的文献阅读计划。

**使用：** 创建阅读清单管理进度
**兼容性：** Zotero 6.x/7.x
**应用：** 研究计划、课程阅读管理

### 17. Zotero annotation manage 0.7.38
Annotation Manage插件专门用于管理PDF中的注释和标注，提供集中的注释管理界面。它能够自动提取PDF中的所有注释，按文献或主题组织注释，提供强大的注释搜索功能，并支持将注释导出为可用的格式。这个插件使得用户可以集中管理所有文献的阅读笔记，快速查找特定主题的相关注释，为论文写作提取关键观点和引用提供便利。

**使用：** 打开Annotation Manage界面
**兼容性：** Zotero 6.x
**应用：** 阅读笔记管理、注释汇总
